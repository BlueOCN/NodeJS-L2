# NodeJS-L2

## Getting started
Install express locally using node package manager
```PS
npm install express
```

Start a new project
```PS
express red-bicicletas --view=pug
```

Change to app's directory
```PS
cd .\red-bicicletas\
```

Install dependencies
```PS
npm install
```

Install nodemon
```PS
npm install nodemon --save-dev
```

Run the app
```PS
npm start
```

Run the app with nodemon
```PS
npm run devstart
```
